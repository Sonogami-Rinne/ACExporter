# ACExporter

A BepInEx plugin for Aicomi to export characters to PMX.

## How to use
Put the ACExporter.dll to "BepInEx\plugins" 

---

Press Ctrl + Tab in the character maker, then a menu will appear

<img width="652" height="897" alt="屏幕截图 2026-04-21 142532" src="https://github.com/user-attachments/assets/54d616e3-ff8d-4155-80d0-272e9e17328d" />

---

Click the Export button, a folder will pop up after a few minutes. (This takes longer than the time required by the KK_Exporter)

<img width="627" height="771" alt="屏幕截图 2026-04-21 142819" src="https://github.com/user-attachments/assets/943740d7-745c-4c8f-bd2e-cd03ce9f4495" />

---

Then Install the plugin for blender(([SVS branch of KKBP](https://gitgoon.dev/kkbp-dev/KKBP_Importer/src/branch/SVS_Compatible)) to import the model.pmx in the folder 

<img width="347" height="377" alt="屏幕截图 2026-05-02 115018" src="https://github.com/user-attachments/assets/8577f182-a329-4a1c-8567-ce96cb122866" />

---

After minutes, the model will appear

<img width="706" height="842" alt="屏幕截图 2026-05-02 115148" src="https://github.com/user-attachments/assets/9e3732a2-4a85-451a-9582-a4d4e6122191" />

## Other Export Options

### ExportAllOutfits
One click for exporting all outfit variations of current character

### ExportWithMainCamera
An experimental function.This will take more time but export better textures.
